Build the source code

Verdaccio relies on yarn instead npm to download depenedencies.

Note: the current build only will build with ➜ yarn latest.

  yarn install

Scripts

We have a list of scripts that you will use for diferent kind of tasks, in the following section we describe all posible task based on branches.

Branch (2.x)

On branch 2.x the unique part we have to build is the UI which is based on React.js, webpack and CSS Modules.

Scripts

scriptDescription
releasethis script is used to generate changelog and raise up the version according the commits messages
prepublishit ensures before publish the new ui is being generated
testrun all the test
pre:cispecific task for CI, build the UI required for test
test:cirun test generating coverage
test:onlyrun only test
test:coveragerun nyc as a wrapper to generate coverage with mocha test
coverage:htmlrun nyc to generate coverage reports
coverage:publishpublish on codecov the coverage (don't use it)
lintrun the linting for javascript code.
lint:cssrun the linter for css
dev:webuirun a webpack server with hot reloading enabled http://localhost:4872/#/ it requires a verdaccio server running in port 4873.
pre:webpackprepare the field for webpack (it a substask of build:webui)
build:webuicreate the static assets for the UI with webpack
build:dockercreate a local docker image with verdaccio
build:rpicreate a local docker for raspberry pi image with verdaccio (experimental with no support)
