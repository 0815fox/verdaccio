Fácil de Instalar

$> npm install --global verdaccio 
$> yarn global add verdaccio

Fácil de Configurar

$> npm set registry http://localhost:4873 
$> npm adduser --registry http://localhost:4873

Fácil de Usar

 $> verdaccio 
 warn --- config file  - /home/.config/verdaccio/config.yaml 
 warn --- http address - http://localhost:4873/ - verdaccio/3.0.0

Eso es todo ! Disfruta tu registro privado.

Los clientes npm mas populares son compatibles

npm, yarn y pnpm son parte del ambiente de desarrollo e intentamos mantenernos al día con las últimas actualizaciones.

Facilitando el trabajo de los DevOps

Ofrecemos una imagen oficial en Docker lista para usar 

docker pull verdaccio/verdaccio

y soporte de Kubernetes Helm para fácil despliegue

helm install --name npm stable/verdaccio

Soporte de Extensiones

Verdaccio esta basado en extensiones, autenticación, soporte para middleware y almacenamiento. Solo elige uno o crea el tuyo propio.
